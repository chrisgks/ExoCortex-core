# Claude Bridge

This repository is operated through ExoCortex wrapped sessions.

If a wrapped ExoCortex bootstrap prompt is present, it is authoritative for this session.

Before substantive work, read the ExoCortex files surfaced by that bootstrap, starting with the most specific relevant scope.

If you are not in a wrapped session, read these files first:

- `README.md`
- `AGENT.md`
- `STATE.md`
- `MEMORY.md`
- `DECISION RULES.md`
- `SKILLS.md`
- `wiki/index.md`
- `wiki/00_meta/Scope.md`
- relevant files under `system/`

Contracts are lazy: any file in this list that is absent is simply skipped.

`AGENTS.md` in this repo is a wiki-maintenance contract, not the full ExoCortex runtime contract by itself.
