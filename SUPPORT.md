# Support

## Getting Started

1. Run `python3 tools/bootstrap/init.py --install-wrappers`.
2. Open a fresh shell and run `exocortex-doctor`.
3. Start from the repo root or a narrower context such as `domains/work/`.

## Where To Ask For Help

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Use GitHub Discussions, if enabled, for open-ended questions about setup, workflow, and design tradeoffs.

## Before Opening An Issue

- Run `python3 -m unittest discover -s tests`.
- Confirm the problem still exists in a clean clone or in the exported public repo.
- Check whether the issue is really about the public core versus your private instance data.
