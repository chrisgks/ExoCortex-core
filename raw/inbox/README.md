# Inbox

`raw/inbox/` is the pre-ingestion holding area for material that has not yet been promoted into a clearer pipeline.

Use these buckets:

- `concept_seeds/`: interesting thoughts that may later become writing, memory, or wiki material
- `design_notes/`: internal architecture or system notes that may later become source notes or design pages
- top-level inbox: temporary drop zone only; files should be categorized out of it quickly

Do not ingest directly from a mixed inbox when the file type is still unclear. Categorize first, then decide whether the next destination is:

- a writing project
- `raw/sources/`
- a scoped wiki
- or continued holding in `raw/inbox/`
