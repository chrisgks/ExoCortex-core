What this proves

The wrapper layer is real.

These are live outputs from this repo, not invented CLI text. The system really resolves wrapped harnesses and really assembles an authoritative preload.

Why it matters
  • You do not start cold.
  • You can inspect exactly what context got loaded.
  • The repo can evolve through ordinary use rather than isolated prompts.
Founding constraint

Built for one real brain first.

This project started as infrastructure for functioning better in the world, not as a generic AI framework. That is why it privileges explicit contracts, durable context, and recoverable state.
