# System Control Plane

The `system/` folder holds control-plane files for the outer loop.

Use this folder for:

- self-modeling
- persona calibration
- health-state adaptation
- routing and questioning policy
- system-level decision rules
- intelligence-loop triggers
- intended behaviour contracts
- usage-rate tables for token and dollar-cost accounting
- cron-job registry
- review cadence
- global priorities and open loops

This folder is not a domain. It is the operating layer above domains and projects.
