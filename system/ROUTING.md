# Routing

## Root Default

- At repository root, use `chief-of-staff`.
- At repository root, default mode is `conversation` unless explicitly overridden.

## By Domain

- `domains/work/`: usually `planning`, `research`, or `builder`
- `domains/life/`: usually `life-systems`
- `domains/learning/`: usually `research` or `life-systems`
- `domains/writing/`: usually `planning` or `research`

## Routing Rule

Choose the agent that best fits the current mode, not just the topic.
