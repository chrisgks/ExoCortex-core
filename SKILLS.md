# Root Skills

This file maps the root context to the reusable skills shipped in
`skills/`. The skill definitions live in `skills/`; note here which ones
matter at root scope. This starter lists the engine's generic skills;
add your own as they earn a place.

## Available Skills

- `skills/inbox-triage/` - triage files dropped into `raw/inbox/` before wiki ingestion.
- `skills/repo-intake/` - get oriented in an unfamiliar repository.
- `skills/implement-feature/`, `skills/fix-bug/`, `skills/add-tests/`, `skills/debug-failure/` - core engineering loops.
- `skills/code-review/` - severity-ranked, evidence-backed code review.
- `skills/verification-discovery/` - find and run the checks that prove a change works.
- `skills/grilling/` - relentless one-question-at-a-time interview that resolves every open decision.
- `skills/handoff/` - compact a session into a handoff doc for the next agent; redacts secrets and instance data.
- `skills/git-guardrails/` - pre-commit hook that blocks instance paths and secrets from a public repo.
- `skills/writing-great-skills/` - house style for authoring predictable, well-edited SKILL.md files.
