# Changelog

All notable changes to the public ExoCortex core should be documented in this file.

## [0.1.0] - 2026-04-13

First public release.

### Added

- context-aware wrappers for `codex`, `claude`, and `gemini`
- session capture, journaling, summary extraction, and promotion workers
- clean-slate bootstrap via `exocortex-init` and `tools/bootstrap/init.py`
- starter root, system, domain, journal, raw, and wiki scaffolds
- reusable agents, skills, templates, and tests
- exploratory Mission Control source

### Notes

- This release is intentionally local-first and file-first.
- The public repo ships clean scaffolds, not anyone's private operating history.
